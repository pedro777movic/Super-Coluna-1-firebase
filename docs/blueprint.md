# **App Name**: Super Coluna

## Core Features:

- Dynamic Sequential Page Architecture: An 18-section high-conversion layout designed for mobile-first scrolling with structural placeholders for logic and emotional journey stages.
- Progressive Diagnostic Quiz Framework: Interactive UI container with a functional progress bar and step-based quiz state management for cold traffic identification.
- AI Path Analysis Tool: Generative AI tool that processes placeholder diagnostic data from the quiz to generate a unique visual recovery overview narrative for the user.
- SaaS Premium Dashboard Mockups: Component library specifically for health-app visuals including Home, Evolution, Achievements, and Profile cards with smooth rounded corners.
- Interactive Health Timelines: A visual progression component used for amplifying problem consequences and showcasing the 'Super Coluna' method steps with animated transitions.
- Lightweight Animation System: Optimized micro-interactions and scroll-triggered fade-ins using Framer Motion to ensure high performance on PageSpeed Mobile.
- Global User Preference Store: Persistent state management for user results across the 18 sections for personalized landing page sections without requiring a dedicated Firestore setup.

## Style Guidelines:

- Primary Color: Deep Safety Blue (#29719F) for trust and reliability in spine health. Secondary: Healing Green (#4CAF50) to signify mobility and vitality.
- Backgrounds: Clean Arctic White (#FFFFFF) and a Soft Ghost Grey (#F5F7FA) for alternate sections, ensuring a minimalist health-tech vibe.
- Font Pairing: 'Poppins' for headlines to give a contemporary geometric feel; 'Inter' for body text to maintain an objective, machine-precision look. San-serif pairing.
- Icon style: Thin-line, minimalist healthcare icons with rounded stroke endings to avoid a harsh hospital or gym appearance.
- Mobile-first, card-driven interface with 16px to 24px margins, generous negative space, and elegant box shadows (0px 10px 30px rgba(0,0,0,0.05)).
- Soft micro-animations including staggered Fade-In-Up for cards and subtle width expansion on the quiz progress bar.
